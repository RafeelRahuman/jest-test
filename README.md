# Jest Testing Practice

This project contains JavaScript functions tested using Jest.

## 📚 Functions Implemented

1. **Capitalize**
   - Capitalizes the first letter of a string.

2. **Reverse String**
   - Reverses a given string.

3. **Calculator**
   - add
   - subtract
   - multiply
   - divide

4. **Caesar Cipher**
   - Shifts characters in the alphabet
   - Preserves uppercase letters
   - Keeps punctuation unchanged

5. **Analyze Array**
   - Returns:
     - average
     - min
     - max
     - length
